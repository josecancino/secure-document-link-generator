# Secure Document Link Generator

A secure way to share sensitive documents using one-time or time-limited links. Built with **Bun + NestJS + React**.

## Prerequisites

Ensure you have the following installed:

*   **Bun**: `curl -fsSL https://bun.sh/install | bash`
*   **Git**: [git-scm.com](https://git-scm.com/) (to clone the repository)

## Running the Application

To run the application locally, open two terminal tabs/windows:

### 1. Start Backend
```bash
cd backend
bun start:dev
```
*   Server runs on: `http://localhost:3000`

### 2. Start Frontend
```bash
cd frontend
bun run dev
```
*   Client runs on: `http://localhost:5173` (or the next available port)
